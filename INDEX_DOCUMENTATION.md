## 📑 Quick Command Reference

```bash
# Setup (from SETUP_GUIDE.md)
composer install && npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed --class=PemilihanSeeder
npm run build
php artisan serve

# Development (from QUICK_START.md)
npm run dev          # Watch CSS/JS
php artisan serve    # Run server

# Troubleshooting (from CONFIG_REFERENCE.md)
php artisan cache:clear
php artisan route:clear
composer dump-autoload
php artisan migrate:fresh --seed

# Database (from CONFIG_REFERENCE.md)
php artisan tinker                    # Interactive shell
mysql -u root pemilihan-osis         # MySQL shell
php artisan db:seed --class=PemilihanSeeder  # Seed data
```

## 📌 Important Reminders

### ⚠️ Before Reading
- You need PHP 8.2+, MySQL, Node.js 18+
- Check that laragon/XAMPP/MAMP is running
- Have a code editor ready (VS Code recommended)

### ⚠️ During Setup
- Don't skip migration: `php artisan migrate`
- Don't forget to seed: `php artisan db:seed`
- Do build CSS: `npm run build`
- Keep terminal open for `php artisan serve`

### ⚠️ During Development
- Use `@csrf` on all forms
- Check console for errors (F12)
- Test responsive (F12 → toggle device)
- Commit regularly with git

### ⚠️ Before Deploying
- Set `APP_DEBUG=false`
- Use HTTPS (not HTTP)
- Configure email variables
- Backup database first
- Test all pages in production
