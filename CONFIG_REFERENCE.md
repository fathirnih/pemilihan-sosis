## Authentication & Security

### Session Configuration
```php
// config/session.php
'driver' => 'file',        // Store sessions on disk
'lifetime' => 120,         // 120 minutes
'expire_on_close' => false,// Don't expire on browser close
'encrypt' => false,        // Don't encrypt session data
'secure' => false,         // false for local (http), true for production (https)
'http_only' => true,       // Don't allow JS access
'same_site' => 'lax',      // CSRF protection
```

### Cookie Configuration
```php
// config/session.php
'cookie' => 'PEMILIHAN_OSIS',  // Cookie name
'path' => '/',                 // Cookie path
'domain' => null,              // Will use current domain
'secure' => false,             // true for https only
'http_only' => true,           // No JS access
'same_site' => 'lax',          // CSRF protection
```

## Model Configuration

### TokenPemilih Model
```php
// app/Models/TokenPemilih.php
namespace App\Models;

class TokenPemilih extends Model
{
    protected $fillable = [
        'periode_id',
        'tipe_pemilih',    // 'siswa' or 'guru'
        'pemilih_id',      // ID of siswa/guru
        'token',           // Unique login token
        'status',          // 'aktif', 'digunakan', 'kadaluarsa'
        'sudah_memilih',   // Boolean
    ];
    
    protected function casts(): array
    {
        return [
            'sudah_memilih' => 'boolean',
            'created_at' => 'datetime',
            'updated_at' => 'datetime',
        ];
    }
}
```

### Key Relationships
```php
// TokenPemilih
$token->periode();     // BelongsTo PeriodePemilihan
$token->suara();       // HasOne Suara

// PeriodePemilihan
$periode->kandidat();  // HasMany Kandidat
$periode->token();     // HasMany TokenPemilih
$periode->suara();     // HasMany Suara

// Kandidat
$kandidat->anggota();  // HasMany KandidatAnggota
$kandidat->suara();    // HasMany Suara

// Suara
$suara->periode();     // BelongsTo PeriodePemilihan
$suara->kandidat();    // BelongsTo Kandidat
```

## Performance Tuning

### Eager Loading (prevent N+1 queries)
```php
// Good ✅
$kandidats = $periode->kandidat()->with('anggota.siswa', 'suara')->get();

// Bad ❌ (causes N+1)
$kandidats = $periode->kandidat()->get();
// Then accessing $kandidat->anggota for each causes individual query
```

### Query Optimization
```php
// Use select() to limit columns
Kandidat::select('id', 'nomor_urut', 'nama')->get();

// Use chunk() for large datasets
Suara::chunk(100, function ($votes) {
    // Process 100 votes at a time
});

// Use paginate() for limiting results
Kandidat::paginate(15);
```

### Cache Results
```php
// Cache results for 1 hour
$results = cache()->remember('voting_results', 3600, function () {
    return Kandidat::with('suara')->get();
});
```

## Troubleshooting

### "Token mismatch" Error
- CSRF protection triggered
- Solution: Clear browser cookies, ensure `@csrf` in forms

### "Class not found" Error
- Autoloader cache issue
- Solution: `composer dump-autoload`

### "Route not found" Error
- Route cache stale
- Solution: `php artisan route:clear`

### "View not found" Error
- Check file path and capitalization
- Ensure `.blade.php` extension

### "Migration already exists" Error
- Duplicate migration timestamp
- Solution: Adjust migration timestamp, or use fresh: `php artisan migrate:fresh`

### Database Connection Refused
- Check `.env` credentials
- Verify MySQL running
- Check firewall rules
- Verify port 3306 accessible
